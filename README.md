# Number Guessing Game

## Overview
The Number Guessing Game is a simple web application that challenges users to guess a randomly generated number between 1 and 100. The game provides feedback on each guess, indicating whether the guess is too high, too low, or correct. 

## Features
- Random number generation between 1 and 100.
- User-friendly interface for inputting guesses.
- Feedback messages to guide the user.
- Count of attempts made by the user.
- Modern dark theme for an enhanced visual experience.

## Files
- `src/index.html`: Contains the structure of the web application.
- `src/style.css`: Provides styling with a modern dark theme.
- `src/script.js`: Implements the game logic and user interaction.

## How to Run the Game
1. Clone the repository to your local machine.
2. Open the `src/index.html` file in a web browser.
3. Start guessing the number!

## Additional Notes
- Ensure that JavaScript is enabled in your browser for the game to function correctly.
- Refresh the page to start a new game.
